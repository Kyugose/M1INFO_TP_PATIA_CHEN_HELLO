Le fichier Test_perfm.py lance les solveur sur les puzzle présent dans le répertoire puzzle et compare les performance de chaque algo bfs dfs et astar 
Test_perfm.py ne prend pas d'argument
Sachant que il y'a un time out de 5 s pour résoudre le puzzle.

Lancer : python Test_perfm.py

Il produit deux plot une avec le total de temps utilisé par chaque algorithme et un avec le temps efféctué pour résoudre un probleme d'un sequence max donné pour chaque algorithme.
Nous avons sauvegarder ces 2 plots dans courbe_performance_temps_total.png et courbe_performance_sequencemax.png.
